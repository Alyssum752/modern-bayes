\frame{
\frametitle{Principles to Teaching Bayesian Statistics}

\begin{enumerate}
\item State expectations and be clear. 
\item Be organized and consistent. 
\item Be excited in what you're teaching. 
\item Teach material that you use in practice. 
\item Advocate good programming practices. 
\item Use motivational examples and real data. 
\item Don't be afraid to say you don't know the answer.
\item Typos. 
\item Be available for your students to answer questions. 
\end{enumerate}



}

\frame{

Let's go through these principles.

\frame{

Teach methods that you use in practice to your students.

\vskip 1em

\begin{itemize}
\item Make the methods accessible. 
\item Example why these methods are important. 
\item Use examples and pictures when you can. 
\end{itemize}


}

\frame{

Advocate for good programming practices. 

\begin{itemize}
\item Reproducible (markdown).
\item Version control (git). 
\item Using a style guide (Google). 
\item Good documentation of code. 
\end{itemize}

Importance of these and give students feedback. 

}

\frame{
\frametitle{Use Motivational Examples and Real Data}

\begin{itemize}
\item Lecture is on hierarchical Bayesian modeling. 
\item A motivation might be data you work with in your research. 
\item You can tie this in throughout the lecture and also in the lab
and homework to reinforce concepts. 
\item Students like working with real data as well and it's good experience! 
\end{itemize}



}

\frame{
\frametitle{Answering Questions}

This part can often be the most tough. 

\vskip 1em
Address questions that you don't know the answer to honestly and that you will get back to the students about it. 



}

\frame{
\frametitle{Typos and Mistakes}

\begin{itemize}
\item Read your lecture notes for typos. 
\item Have a TA proof read your notes for typos. 
\item Even if you have minor typos, you will get complaints. 
\end{itemize}

Suggestions: Ask a student volunteer or TA to write down the typos and state that you will correct them after the class. Deal with them promptly and quickly. 


}

\frame{
\frametitle{Be Available for Questions}

Answer emails promptly and quickly. 

\vskip 1em

Make yourself available for your students and be understanding. 



}
